MARCUS CURRIT — веб-приложение для работы с латинскими текстами: для удобного доступа к библиотеке текстов и для анализа латинских текстов с морфологической разметкой и переводом и возможностью создать т. н.  словарик.


Это веб-приложение на Flask, которое позволяет:  
1. Получать морфологическую информацию о каждом слове текста. Падеж, число, род, время, залог и т.д.  
2. Переводить слова. Автоматический перевод латинских слов на английский  
3. Просматривать библиотеку текстов. Тексты сгруппированы по авторам  

Для реализации веб приложения используется:  
1. Python
2. Flask (для создания веб-приложения)
3. Stanza (для получения морфологического разбора)
4. Latino (для получения перевода латинских слов)
5. python-slugify (для создания ссылок)
6*. урезананя версия веб-приложения (только просмотре библиотеки без возсожности морфологического разбора) размещается на PythonAnywhere.
   

Описание элементов проекта:  
app.py:  
sent_info(word). Анализирует одно предложение: извлекает лемму, часть речи, грамматические признаки, перевод  
text_info(word). Анализирует весь текст по предложениям  
parse_text_file(filepath). Парсит файлы текстов (извлекает данные из заголовка)  
load_all_texts(). Кэшированная загрузка всех текстов из папки texts/ 


@app.route('/'). Главная страница  
@app.route('/done_dict'). Обработка формы анализа текста  
@app.route('/library'). Список всех авторов  
@app.route('/library/<author_slug>'). Тексты конкретного автора  
@app.route('/library/<author_slug>/<text_id>'). Просмотр отдельного текста  

HTML-шаблоны  
main.html. Главная страница с формой для ввода текста  
done_d.html. Отображение результатов морфологического анализа  
authors.html. Список авторов с количеством текстов  
texts_by_author.html. Список текстов выбранного автора  
text_view.html. Полное отображение текста с метаданными
